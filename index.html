23.MR Request Approval

Warehouse users can approve or disapprove the request raised from the customer end based on the available stock. The system will automatically by default approve the MR request whose cost is 20k. The system will send an email notification for MR request approval where the Total item cost is more than 20k to the respective warehouse authority.
How to Approve MR Request?
To approve MR request follow the below steps:
1.Click on the “MR Request Approval” sub menu.
2.Click on the Pending radio Button on top of the page
3.Click on the Approval button in the Action column to approve the request.
